## 문제 1번

### 문제 내용

정수 n을 입력받아 1부터 n까지 중 n의 약수를 모두 출력하고, 약수의 개수도 함께 출력하라.

```
입력: 12
약수: 1 2 3 4 6 12
개수: 6
```

### 소스코드

```c
#include <stdio.h>

int main(void) {
    int n, count = 0;
    printf("입력: ");
    scanf("%d", &n);

    printf("약수: ");
    for (int i = 1; i <= n; i++) {
        if (n % i == 0) {
            printf("%d ", i);
            count++;
        }
    }
    printf("\n개수: %d\n", count);
    return 0;
}
```

### 실행화면

<!-- 예: ![실행화면](./screenshots/week02_1_실행화면.png) -->

```

약수: 1 2 3 4 6 12
개수: 6
```

### 피드백

**어려움을 겪었던 부분**  
처음에 i를 0부터 시작해서 0으로 나누기 오류가 발생했다. 1부터 시작해야 한다는 걸 몰랐다.

**문제를 통해 배우게 된 내용**  
`%` 연산자로 나머지를 구해서 약수를 판별할 수 있다는 것을 배웠다. 카운터 변수를 따로 두어 개수를 세는 방법도 익혔다.

## 문제 2번

### 문제 내용

정수 n을 입력받아 소수인지 아닌지 출력하라.

```
입력: 7  →  소수입니다
입력: 9  →  소수가 아닙니다
```

### 소스코드

```c
#include <stdio.h>

int main(void) {
    int n, count = 0;
    printf("입력: ");
    scanf("%d", &n);

    for (int i = 1; i <= n; i++) {
        if (n % i == 0)
            count++;
    }

    if (count == 2)
        printf("소수입니다\n");
    else
        printf("소수가 아닙니다\n");

    return 0;
}
```

### 실행화면

<!-- ![실행화면](./screenshots/week02_2_실행화면.png) -->

```
입력: 7
소수입니다
```

### 피드백

**어려움을 겪었던 부분**  
소수의 정의가 "약수가 1과 자기 자신뿐인 수"인 건 알았지만, 코드로 어떻게 표현할지 몰랐다. 힌트에서 약수 개수가 2개이면 소수라는 것을 보고 이해했다.

**문제를 통해 배우게 된 내용**  
1번 문제의 약수 판별 로직을 그대로 활용해서 소수를 판별할 수 있다는 것을 배웠다. 이전 문제의 코드를 재활용하는 방식을 익혔다.

## 문제 3번

### 문제 내용

정수 n을 입력받아 2부터 n까지의 소수를 모두 출력하고, 개수도 함께 출력하라.

```
입력: 20
소수: 2 3 5 7 11 13 17 19
개수: 8
```

### 소스코드

```c
#include <stdio.h>

int main(void) {
    int n, total = 0;
    printf("입력: ");
    scanf("%d", &n);

    printf("소수: ");
    for (int i = 2; i <= n; i++) {
        int count = 0;
        for (int j = 1; j <= i; j++) {
            if (i % j == 0)
                count++;
        }
        if (count == 2) {
            printf("%d ", i);
            total++;
        }
    }
    printf("\n개수: %d\n", total);
    return 0;
}
```

### 실행화면

<!-- ![실행화면](./screenshots/week02_3_실행화면.png) -->

```
입력: 20
소수: 2 3 5 7 11 13 17 19
개수: 8
```

### 피드백

**어려움을 겪었던 부분**  
2번 소수 판별 로직을 안쪽 for에 넣는 것이 헷갈렸다. 변수 이름이 겹치지 않도록 count와 total을 따로 쓰는 부분에서 실수했다.

**문제를 통해 배우게 된 내용**  
중첩 for문을 활용해 여러 수에 대해 반복적으로 판별하는 방법을 배웠다. 바깥 반복문은 대상 수를 순회하고, 안쪽 반복문은 약수를 세는 역할을 한다는 구조를 이해했다.

## 문제 4번

### 문제 내용

소수를 몇 개 찾을지 입력받아, 그 개수만큼의 소수를 순서대로 출력하라.

```
입력: 5
출력: 2 3 5 7 11
```

### 소스코드

```c
#include <stdio.h>

int main(void) {
    int target, found = 0, num = 2;
    printf("입력: ");
    scanf("%d", &target);

    printf("출력: ");
    while (found < target) {
        int count = 0;
        for (int i = 1; i <= num; i++) {
            if (num % i == 0)
                count++;
        }
        if (count == 2) {
            printf("%d ", num);
            found++;
        }
        num++;
    }
    printf("\n");
    return 0;
}
```

### 실행화면

<!-- ![실행화면](./screenshots/week02_4_실행화면.png) -->

```
입력: 5
출력: 2 3 5 7 11
```

### 피드백

**어려움을 겪었던 부분**  
3번과 달리 범위가 정해지지 않아서 for 대신 while을 써야 한다는 것을 처음엔 몰랐다. 언제 멈출지 기준을 찾은 수의 개수로 잡는 발상이 어려웠다.

**문제를 통해 배우게 된 내용**  
반복 횟수가 아니라 조건(찾은 소수 개수)으로 반복을 제어할 때 while이 적합하다는 것을 배웠다. 범위가 정해지지 않은 탐색 문제에서는 while + 카운터 조합이 유용하다는 것을 익혔다.
