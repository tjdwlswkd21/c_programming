# Week 3 문제 모음

멤버별로 문제가 다릅니다. 본인 이름의 문제를 풀어주세요.

주제: **함수 형태 다양화 · break/continue · 변수 범위와 존재기간 · switch문**

---

### 함수 형태 참고

```c
반환형   함수이름(매개변수형 매개변수명) { ... }

void     f(void)          // 입력도 없고 반환도 없음
int      f(void)          // 입력 없음, 정수 반환
void     f(int n)         // 정수 하나 입력, 반환 없음
int      f(int a, int b)  // 정수 둘 입력, 정수 반환
double   f(int a, int b)  // 정수 둘 입력, 실수 반환
```

---

## 공통 (전원 동일)

### 1번

매개변수도 없고 반환값도 없는 안내 출력 함수를 작성하라.  
`main` 시작 시 가장 먼저 호출해 프로그램 이름과 사용법을 출력하라.

```c
void 안내(void) { ... }
```

```
=== 계산기 프로그램 ===
두 정수와 연산 번호를 입력하세요.
1: 덧셈  2: 뺄셈  3: 곱셈  4: 나눗셈
```

> 힌트: 함수 본문에 `printf`만 있으면 됨. `return`은 생략 가능.

### 2번

두 정수 a, b와 연산 번호를 매개변수로 받아 결과를 `double`로 반환하는 계산기 함수를 작성하라.  
`main`에서 세 값을 입력받아 함수를 호출하고 결과를 출력하라.

```c
double 계산기(int a, int b, int 연산) { ... }
```

| 연산 번호 | 동작 |
|------|------|
| 1 | a + b |
| 2 | a - b |
| 3 | a × b |
| 4 | a ÷ b |
| 그 외 | 0.0 반환 |

```
입력: 10 3 4
결과: 3.33
```

> 힌트:
> - `switch (연산)` 으로 분기. `default`도 반드시 작성.
> - 정수끼리 나누면 소수점이 잘림 — `(double)a / b` 로 형변환 필요.
> - b가 0이면 0.0을 반환.

## 강민채

### 1번

정수를 계속 입력받는다. **양수면 누적**하고, **음수면 건너뛰고**(continue), **0이면 입력 종료**(break). 누적된 양수의 합을 반환하는 함수를 작성하라.

```c
int 양수합(void) { ... }
```

```
입력: 3 -1 5 -2 0
출력: 8
```

> 힌트:
> - 매개변수 없이 함수 내부에서 `scanf`로 직접 입력받기.
> - 음수 조건에서 `continue`, 0 조건에서 `break`.
> - 합을 담을 변수는 함수 안 지역변수로 선언.

### 2번

함수를 호출할 때마다 **지금까지의 호출 횟수**를 반환하는 함수를 작성하라.  
`main`에서 5번 호출해 매번 반환값을 출력하라.

```c
int 호출횟수(void) { ... }
```

```
1번째 호출
2번째 호출
3번째 호출
4번째 호출
5번째 호출
```

> 힌트:
> - **static** 지역변수를 사용하면 함수가 끝나도 값이 유지됨.
>   ```c
>   static int 횟수 = 0;
>   횟수++;
>   return 횟수;
>   ```
> - **전역변수를 피해야 하는 이유**: 전역변수는 파일 안의 모든 함수에서 수정할 수 있어, 어느 함수가 값을 바꿨는지 추적하기 어렵다. 함수가 많아질수록 버그를 찾기 어려워지므로, 전역변수는 되도록이면 최소화할 것.

### 3번

양의 정수 n을 받아 정수를 n번 입력받으면서 **최대값과 최소값을 추적**해 마지막에 출력하는 함수를 작성하라. 전역변수 없이 지역변수만 사용하라.

```c
void 최대최소(int n) { ... }
```

```
입력: 5
3 7 1 9 2
최대: 9  최소: 1
```

> 힌트:
> - 최대값·최소값 변수를 첫 번째 입력값으로 초기화하고, 이후 입력마다 비교해서 갱신.
> - 지역변수로 선언하면 함수가 끝나면 사라짐 — 이것이 정상.

## 김민석

### 1번

양의 정수 n을 받아 **n의 가장 작은 약수(1 제외)를 반환**하는 함수를 작성하라.  
약수를 찾는 순간 `break`로 탐색을 중단하라. n이 소수면 n 자체를 반환하라.

```c
int 최소약수(int n) { ... }
```

```
입력: 12  →  2
입력: 7   →  7
```

> 힌트:
> - 2부터 n-1까지 반복하며 `n % i == 0` 확인.
> - 약수를 찾으면 결과 변수에 저장하고 `break`.
> - 반복 종료 후 결과 변수가 초기값이면 소수 → n을 반환.

### 2번

양의 정수 n을 받아 1부터 n까지 반복하면서 **3의 배수는 건너뛰고**(continue), 나머지 수들의 합을 반환하는 함수를 작성하라.

```c
int 합산(int n) { ... }
```

```
입력: 10
출력: 37  (1+2+4+5+7+8+10)
```

> 힌트:
> - `if (i % 3 == 0) continue;` 로 3의 배수 건너뛰기.
> - `continue` 이후의 코드(합산)는 실행되지 않음.

### 3번

정수 하나를 매개변수로 받아 **지금까지 받은 모든 값의 누적합을 반환**하는 함수를 작성하라.  
`main`에서 5번 호출해 매번 반환값을 출력하라.

```c
int 누적합(int 값) { ... }
```

```
입력(5번): 10 20 5 3 2
10
30
35
38
40
```

> 힌트:
> - **static** 지역변수로 합을 유지.
>   ```c
>   static int 합 = 0;
>   합 += 값;
>   return 합;
>   ```
> - **전역변수를 피해야 하는 이유**: 전역변수는 파일 안의 모든 함수에서 수정할 수 있어, 어느 함수가 값을 바꿨는지 추적하기 어렵다. 함수가 많아질수록 버그를 찾기 어려워지므로, 전역변수는 되도록이면 최소화할 것.

## 오병욱

### 1번

구구단 n단을 출력하되, **곱이 처음으로 한계값을 초과하는 순간 출력을 중단**(break)하는 함수를 작성하라.

```c
void 구구단제한(int n, int 한계) { ... }
```

```
입력: 7 30
7 × 1 = 7
7 × 2 = 14
7 × 3 = 21
7 × 4 = 28
```

> 힌트:
> - `for (곱하는수 = 1; 곱하는수 <= 9; 곱하는수++)` 반복 안에서 곱 계산.
> - 곱이 한계 초과면 `break`.

### 2번

양의 정수 n을 받아 1부터 n까지 반복하면서 **홀수는 건너뛰고**(continue), 짝수만 출력하는 함수를 작성하라.

```c
void 짝수출력(int n) { ... }
```

```
입력: 10
2 4 6 8 10
```

> 힌트:
> - `if (i % 2 != 0) continue;` 로 홀수 건너뛰기.
> - `continue` 이후의 출력 코드는 실행되지 않음.

### 3번

정수 하나를 매개변수로 받아 **지금까지 입력된 수 중 최대값을 반환**하는 함수를 작성하라.  
`main`에서 5번 호출해 매번 현재 최대값을 출력하라.

```c
int 최대값(int 값) { ... }
```

```
입력(5번): 3 7 2 9 1
현재 최대: 3
현재 최대: 7
현재 최대: 7
현재 최대: 9
현재 최대: 9
```

> 힌트:
> - **static** 지역변수로 최대값을 유지.
>   ```c
>   static int 최대 = ...;  // 첫 번째 값을 어떻게 처리할지 고민하기
>   ```
> - **전역변수를 피해야 하는 이유**: 전역변수는 파일 안의 모든 함수에서 수정할 수 있어, 어느 함수가 값을 바꿨는지 추적하기 어렵다. 함수가 많아질수록 버그를 찾기 어려워지므로, 전역변수는 되도록이면 최소화할 것.
