## Ch03. 증감 연산자

### 힌트

- **전치 (`++a`)** : 증가 **후** 사용
- **후치 (`a++`)** : 사용 **후** 증가
- 증감 연산자는 **변수**(메모리 주소가 있는 것)에만 사용 가능
- 리터럴 상수(`5`)나 수식 결과(`a+b`)는 값만 있고 주소가 없으므로 불가

### 문제

`int a = 5, b = 5` 선언 후:

1. `printf("%d", a++)` 와 `printf("%d", ++b)` 실행 전 예상값을 주석으로 적고, 실행해서 비교하라.

2. `--a`, `b--` 도 동일하게 반복하라.

3. 아래 코드가 왜 컴파일 오류인지 주석으로 설명하라.
   ```c
   // 5++;
   // (a + b)++;
   ```

## Ch04. 데이터 표현 방식 + scanf

### 힌트

- 메모리에는 **비트만** 저장된다
- `%d`, `%o`, `%x` 는 같은 비트를 **다르게 해석하는 렌즈**
- 비트 연산자는 그 비트를 직접 조작하는 도구
- **마스킹**: 특정 비트를 켜거나(`|`) 끄는(`&`) 패턴
- `0b`, `0x` 리터럴 표기법도 확인할 것

### 문제

`int x` 를 `scanf` 로 입력받아:

1. `%d`, `%o`, `%x` 로 각각 출력하라. (같은 비트, 다른 해석)

2. 임의 정수 하나를 추가 선언해 `&`, `|`, `^`, `~`, `<<1`, `>>1` 비트 연산 후 결과를 `%d` 와 `%x` 두 형식으로 출력하라.

3. `unsigned char c = 0b00001111;` 에서 **4번 비트를 켜고**, **0번 비트를 끄는** 마스킹 연산을 수행하라.

## Ch05. 상수와 기본 자료형

### 힌트

- `#define` : 전처리기 치환 (타입 없음)
- `const` : 타입 있는 상수
- `char` 는 내부적으로 정수 — `%c` 는 ASCII 테이블로 문자 출력, `%d` 는 숫자 출력
- 대소문자 ASCII 차이는 **32 고정**

### 문제

1. `#define` 과 `const` 각각으로 상수를 하나씩 정의하고 사용하라.

2. `char c = 'A'` 를 `%c` 와 `%d` 로 출력하고, `int n = 97` 을 `%c` 로 출력하라.

3. 소문자 알파벳 하나를 입력받아 대문자로 변환해 출력하라.
   - `toupper()` 금지, ASCII 산술 이용

## Ch06. printf/scanf 특수문자 + 문자열

### 힌트

- 이스케이프 시퀀스 (`\n`, `\t`, `\\`, `\"`) : 출력 흐름 제어
- 필드폭: `%5d` (우정렬), `%-5d` (좌정렬), `%05d` (빈자리 0 채움), `%.2f` (소수점 자리)
- 히스토그램: 배열 인덱스를 문자 ASCII 값으로 활용하는 패턴 (`count[c]++`)

### 문제

1. `\n`, `\t`, `\\`, `\"` 가 포함된 문장을 `printf` 로 출력하라.

2. `%5d`, `%-5d`, `%05d`, `%.2f` 를 사용해 숫자 3개를 표처럼 정렬 출력하라.

3. 영어 단어 하나를 입력받아 각 문자의 등장 횟수를 `*` 로 히스토그램 출력하라.

   **출력 예시** (입력: `hello`)
   ```
   h: *
   e: *
   l: **
   o: *
   ```

## Ch07. 반복문

### 힌트

- `while` : 종료 조건이 **불규칙**할 때
- `for` : 횟수가 **정해질** 때
- 중첩 루프: 바깥 루프 = 행, 안쪽 루프 = 열 구조로 생각할 것
- 루프 변수 초기화 위치 주의

### 문제

1. `while`: 양수를 계속 입력받아 합산하다가 음수 입력 시 종료, 합계 출력

2. `for`: 1부터 n까지 합 출력

3. 중첩 `for`: 5×5 곱셈표 (행/열 레이블 포함) 출력

## Ch08. 조건 분기

### 힌트

- `&&` : 양쪽 모두 참이어야 참
- `||` : 하나만 참이어도 참
- `if-else if-else` 는 위에서부터 순서대로 체크되므로 **범위 조건은 순서가 중요**
- 입력 유효성 검사는 항상 **가장 먼저** 처리

### 문제

1. 나이와 점수를 입력받아:

   | 조건 | 출력 |
   |------|------|
   | 나이 18 이상 **&&** 점수 60 이상 | `"합격"` |
   | 나이 18 미만 **\|\|** 점수 40 미만 | `"재시험"` |
   | 나머지 | `"보류"` |

2. 점수만 따로 입력받아 `if-else if-else` 로 학점 출력:
   - 점수가 0 이상 100 이하 범위를 벗어나면 `&&` 조건으로 `"유효하지 않은 점수"` 먼저 처리
   - 90 이상 → A
   - 80 이상 → B
   - 70 이상 → C
   - 60 이상 → D
   - 나머지 → F
