## 강민채

### 1번

정수 n을 입력받아 1부터 n까지 중 n의 약수를 모두 출력하고, 약수의 개수도 함께 출력하라.

```
입력: 12
약수: 1 2 3 4 6 12
개수: 6
```

> 힌트: `n % i == 0` 이면 i는 n의 약수. 카운터 변수 하나 추가하기

### 2번

정수 n을 입력받아 소수인지 아닌지 출력하라.

```
입력: 7  →  소수입니다
입력: 9  →  소수가 아닙니다
```

> 힌트: 약수가 1과 자기 자신뿐이면 소수 (약수 개수 == 2)

### 3번

정수 n을 입력받아 2부터 n까지의 소수를 모두 출력하고, 개수도 함께 출력하라.

```
입력: 20
소수: 2 3 5 7 11 13 17 19
개수: 8
```

> 힌트: 2번 판별 로직을 바깥 반복문 안에 넣고 카운터 추가

### 4번

소수를 몇 개 찾을지 입력받아, 그 개수만큼의 소수를 순서대로 출력하라.

```
입력: 5
출력: 2 3 5 7 11
```

> 힌트: while로 소수 개수를 세면서, 목표 개수가 될 때까지 반복

## 김민석 · 오병욱

### 1번

정수 n을 입력받아 높이 n인 역삼각형을 출력하라.

```
입력: 4
****
***
**
*
```

> 힌트: i가 0부터 시작하면 i번째 줄에 (n-i)개의 별 출력. 안쪽 for의 반복 횟수를 i에 따라 바꾸는 것이 핵심

### 2번

정수 n을 입력받아 n×n 테두리만 있는 사각형을 출력하라.

```
입력: 4
****
*  *
*  *
****
```

> 힌트: 행 번호로 첫 줄·마지막 줄 구분 (`i == 0 || i == n-1`), 나머지 줄은 열 번호로 양쪽 끝 구분 (`j == 0 || j == n-1`)

### 3번

정수 n을 입력받아 높이 n인 오른쪽 정렬 직각삼각형을 출력하라.

```
입력: 4
   *
  **
 ***
****
```

> 힌트: i번째 줄에 공백 (n-i)개 먼저 출력한 뒤 별 i개 출력. 공백용 for와 별용 for를 따로 두기

### 4번

두 정수를 입력받아 큰 수를 가로·세로, 작은 수를 안쪽 빈 공간의 한 변으로 하는 속이 빈 직사각형을 출력하라.

```
입력: 5 3
*****
*   *
*   *
*   *
*****
```

> 힌트: 두께 = (바깥 - 안쪽) / 2 를 먼저 구하기. 행/열 번호가 두께 미만이거나 바깥-두께 이상이면 `*`, 나머지는 공백
