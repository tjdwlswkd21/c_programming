주제: **최대공약수 (GCD) · 최소공배수 (LCM)**

---

## 강민채

### 1번

두 양의 정수 a, b를 입력받아 공약수를 모두 출력하고 개수를 출력하라.

```
입력: 12 18
공약수: 1 2 3 6
개수: 4
```

> 힌트: 1부터 a, b 중 더 작은 수까지 반복. `a%i==0 && b%i==0` 이면 공약수. Week 1의 약수 판별(`n%i==0`)을 두 수로 확장한 것.

### 2번

두 양의 정수를 입력받아 최대공약수를 구하라. (유클리드 호제법 사용)

```
입력: 48 18
최대공약수: 6
```

> 힌트:
> ```
> while (나눗수 != 0) {
>     나머지 = 피나눗수 % 나눗수;
>     피나눗수 = 나눗수;
>     나눗수 = 나머지;
> }
> // 반복 종료 후 피나눗수가 최대공약수
> ```

### 3번

두 양의 정수를 입력받아 최소공배수를 구하라.

```
입력: 4 6
최소공배수: 12
```

> 힌트: 최소공배수 = a × b ÷ 최대공약수. 2번 코드를 그대로 활용.

### 4번

정수 n을 입력받아 n의 약수 중 소수인 것만 출력하고 개수를 출력하라.

```
입력: 12
소수인 약수: 2 3
개수: 2
```

> 힌트: Week 1의 약수 나열 코드를 바깥 루프로, 소수 판별 코드를 안쪽 로직으로 중첩하면 됨.
