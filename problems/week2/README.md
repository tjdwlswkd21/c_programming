## 강민채

### 1번

정수 n을 입력받아 1부터 n까지 중 n의 약수를 모두 출력하라.

```
입력: 12
출력: 1 2 3 4 6 12
```

> 힌트: `n % i == 0` 이면 i는 n의 약수

### 2번

정수 n을 입력받아 약수의 개수를 출력하라.

```
입력: 12
출력: 6
```

> 힌트: 1번 코드에서 출력 대신 카운터에 더하기

### 3번

정수 n을 입력받아 소수인지 아닌지 출력하라.

```
입력: 7  →  소수입니다
입력: 9  →  소수가 아닙니다
```

> 힌트: 약수가 1과 자기 자신뿐이면 소수 (약수 개수 == 2)

### 4번

정수 n을 입력받아 2부터 n까지의 소수를 모두 출력하라.

```
입력: 20
출력: 2 3 5 7 11 13 17 19
```

> 힌트: 3번 판별 로직을 바깥 반복문 안에 넣기

## 김민석 · 오병욱

### 1번

정수 n을 입력받아 n×n 꽉 찬 별 사각형을 출력하라.

```
입력: 4
****
****
****
****
```

> 힌트: 바깥 for = 행, 안쪽 for = 열

### 2번

정수 n을 입력받아 n×n 테두리만 있는 사각형을 출력하라.

```
입력: 4
****
*  *
*  *
****
```

> 힌트: 첫 줄·마지막 줄은 꽉 차게, 나머지 줄은 첫 번째와 마지막 열만 `*`

### 3번

정수 n을 입력받아 높이 n인 왼쪽 정렬 직각삼각형을 출력하라.

```
입력: 4
*
**
***
****
```

> 힌트: i번째 줄에 i개의 별 출력

### 4번

두 정수를 입력받아 큰 수를 가로·세로, 작은 수를 안쪽 빈 공간의 한 변으로 하는 속이 빈 직사각형을 출력하라.

```
입력: 6 3
******
*    *
*    *
*    *
*    *
******
```

> 힌트: 2번 코드에서 안쪽 빈 공간 크기만 변수로 바꾸기
