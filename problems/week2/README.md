# Week 2 문제 모음

멤버별로 문제가 다릅니다. 본인 이름의 문제를 풀어주세요.

주제: **함수 · 최대공약수 (GCD) · 최소공배수 (LCM)**

---

### 함수 기본 문법 참고

```c
반환형 함수이름(매개변수형 매개변수명) {
    // 본문
    return 반환값;
}
```

예시:

```c
int 두배(int 값) {
    return 값 * 2;
}

int main(void) {
    printf("%d\n", 두배(5));  // 10
    return 0;
}
```

반환할 값이 없으면 반환형을 `void`로 쓰고 `return;` 또는 생략.

## 강민채

### 1번

두 양의 정수 a, b를 입력받아 공약수를 모두 출력하고 개수를 반환하는 함수를 작성하라.  
`main`에서 함수를 호출해 개수를 출력하라.

```c
int 공약수출력(int a, int b) { ... }
```

```
입력: 12 18
공약수: 1 2 3 6
개수: 4
```

> 힌트: 1부터 a, b 중 더 작은 수까지 반복. `a%i==0 && b%i==0` 이면 공약수. 내부에서 출력하고 개수를 세어 반환.

### 2번

두 양의 정수를 받아 최대공약수를 반환하는 함수를 작성하라. (유클리드 호제법 사용)  
`main`에서 두 수를 입력받아 함수를 호출하고 결과를 출력하라.

```c
int gcd(int a, int b) { ... }
```

```
입력: 48 18
최대공약수: 6
```

> 힌트:
> ```
> while (나눗수 != 0) {
>     나머지 = 피나눗수 % 나눗수;
>     피나눗수 = 나눗수;
>     나눗수 = 나머지;
> }
> return 피나눗수;
> ```

### 3번

두 양의 정수를 받아 최소공배수를 반환하는 함수를 작성하라.  
단, 함수 내부에서 2번의 `gcd` 함수를 호출해 사용하라.

```c
int lcm(int a, int b) { ... }
```

```
입력: 4 6
최소공배수: 12
```

> 힌트: 최소공배수 = a × b ÷ `gcd(a, b)`.

### 4번

정수 n을 받아 n의 약수 중 소수인 것만 출력하고 개수를 반환하는 함수를 작성하라.  
소수 판별은 별도 함수 `int 소수판별(int n)`으로 분리해 작성하라.

```c
int 소수판별(int n) { ... }
int 소수약수출력(int n) { ... }
```

```
입력: 12
소수인 약수: 2 3
개수: 2
```

> 힌트: Week 1의 소수 판별 로직을 `소수판별` 함수로 옮기고, `소수약수출력`에서 약수를 순회하며 `소수판별`을 호출.

## 김민석

### 1번

가로와 세로를 매개변수로 받아 속이 빈 직사각형을 출력하는 함수를 작성하라.  
`main`에서 두 수를 입력받아 함수를 호출하라.

```c
void 직사각형(int 가로, int 세로) { ... }
```

```
입력: 6 4
******
*    *
*    *
******
```

> 힌트: 첫 줄·마지막 줄(`i==0 || i==세로-1`)은 가로 전체 `*`, 나머지 줄은 양 끝(`j==0 || j==가로-1`)만 `*`, 사이는 공백.

### 2번

두 양의 정수를 받아 최대공약수를 반환하는 함수를 작성하라. (유클리드 호제법 사용)  
`main`에서 두 수를 입력받아 함수를 호출하고 결과를 출력하라.

```c
int gcd(int a, int b) { ... }
```

```
입력: 48 18
최대공약수: 6
```

> 힌트:
> ```
> while (나눗수 != 0) {
>     나머지 = 피나눗수 % 나눗수;

### 3번

두 양의 정수를 받아 최소공배수를 반환하는 함수를 작성하라.  
단, 함수 내부에서 2번의 `gcd` 함수를 호출해 사용하라.


> 힌트: 최소공배수 = a × b ÷ `gcd(a, b)`.

### 4번

분자와 분모를 양의 정수로 입력받아 기약분수로 출력하라.  
`gcd` 함수를 호출해 사용하라.

```
입력: 12 18

```

> 힌트: 분자와 분모를 각각 `gcd(분자, 분모)`로 나누면 기약분수.
